# 图书销售系统 - 项目总结

## 系统概述

这是一个功能齐全的现代化Web电商平台，专为图书销售设计。系统采用MVC架构模式，实现了完整的电子商务功能，包括用户管理、图书管理、购物车和订单系统。

## 技术栈

- **前端**: JSP, HTML5, CSS3, JavaScript, Font Awesome图标库
- **后端**: Java Servlet, JavaBean
- **数据库**: MySQL
- **构建工具**: Maven
- **服务器**: Tomcat
- **架构模式**: MVC (Model-View-Controller)

## 主要功能

### 1. 用户系统
- 用户注册/登录/注销
- 个人信息管理
- 用户列表管理（管理员功能）

### 2. 图书系统
- 图书管理（增删改查）
- 图书列表展示
- 图书搜索功能

### 3. 购物系统
- 添加商品到购物车
- 购物车管理（数量修改、删除）
- 订单生成
- 订单管理（查看、管理订单）

### 4. 管理系统
- 用户管理
- 图书管理
- 订单管理

## 特色功能

### 现代化界面设计
- 响应式设计，支持移动端访问
- 渐变色彩和阴影效果
- 图标丰富的用户界面
- 交互动画和悬停效果

### 统计功能
- 实时数据统计
- 动态图表显示
- 数据可视化

### 搜索功能
- 全局搜索功能
- 实时搜索过滤

## 文件结构

- `/src/main/java/beans` - 实体类（User, Book, Cart, Order）
- `/src/main/java/dao` - 数据访问对象类
- `/src/main/java/servlet` - Servlet控制器
- `/src/main/webapp` - JSP视图文件和静态资源
- `/src/main/resources` - 配置文件

## 数据库设计

系统包含以下核心数据表：
- `t_user` - 用户表
- `t_book` - 图书表
- `t_cart` - 购物车表
- `t_order` - 订单表

## 运行方式

1. 确保JDK 8+、Maven和MySQL已安装
2. 导入database.sql创建数据库
3. 配置jdbc.properties中的数据库连接信息
4. 运行 `mvn tomcat7:run`
5. 访问 http://localhost:8086/
